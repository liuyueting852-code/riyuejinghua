Part 01 · 为什么带这款

日间 vs 夜间
配方是怎么分工的

"日月时律"不是起个好听的名字，两支精华的成分是真的按皮肤日夜节律分开设计的

☀ 日间焕活精华液 · 核心诉求
抗氧化 + 提亮 白天防御
凝血酸、四氢姜黄素、山竹果皮提取物、维C衍生物四路抗氧化提亮
🌙 夜间密集精华液 · 核心诉求
密集修护 + 抗初老 夜间修复
十种以上活性肽联用，配合胶原、多糖，主打睡眠时段的深度修护
☀ 抗氧化/提亮体系
凝血酸 + 四氢姜黄素 + 山竹果皮提取物
凝血酸是成分党熟悉的经典提亮成分，四氢姜黄素是姜黄素的稳定衍生物，山竹果皮和LYMO系列的倒捻子素同源
🌙 活性肽矩阵
十种以上肽 + 三肽-1铜（GHK-Cu）+ 棕榈酰五肽-4（经典抗皱肽）
GHK-Cu是护肤界研究最久的铜肽之一，棕榈酰五肽-4是行业内最经典的胶原刺激肽，两个都是有据可查的成分
☀ 质地设计
角鲨烷 + 霍霍巴籽油 + 白池花籽油
轻质油相为主，好吸收不拔干，方便叠加防晒和底妆
🌙 修护+舒缓体系
泛醇 + 尿囊素 + 四氢甲基嘧啶羧酸（依克多因）
依克多因和油霜是同一个成分逻辑，"外部抗应激"贯穿自然源溯多个产品线
☀ 包装区分
橙色瓶身 · 15支
早上洁面护肤后使用，叠加防晒
🌙 包装区分
蓝色瓶身 · 15支
晚上洁面后使用，不用叠加防晒
💡 两支精华都含甘草酸二钾——和光甘草定面膜是同一个"甘草家族"逻辑，抗炎舒缓打底贯穿全系列，这是可以串起来讲的品牌一致性。

Part 03 · 怎么带

弹幕出现什么
直接用哪句话接

不需要背稿，看弹幕找对应场景

「抗初老」「细纹」「紧致吗」
→
✓ 接，讲夜用
「抗初老看夜用这支——十种以上活性肽联用，里面的三肽-1铜（GHK-Cu）是护肤界研究最久的铜肽之一，棕榈酰五肽-4是行业内最经典的胶原刺激肽，晚上睡觉的时候正好是皮肤修复的黄金时间。」 → 追问"这么多肽是不是堆料"：肽类各自分工不同，有的管紧致、有的管修复、有的管舒缓，是协同不是堆砌。
「提亮」「暗沉」「维C」
→
✓ 接，讲日用
「提亮看白天这支——凝血酸+四氢姜黄素+山竹果皮提取物三路一起上，凝血酸是成分党很熟悉的经典提亮成分，四氢姜黄素是姜黄素的稳定衍生物，山竹果皮和LYMO系列的倒捻子素是同一个抗氧化家族。」 → 追问"能美白吗"：这几个成分方向是抗氧化+提亮，具体功效宣称以备案为准，不要自己加码承诺美白。
「早晚都要用吗」「能只用一支吗」
→
✓ 接，讲使用逻辑
「橙色白天用，蓝色晚上用，一盒30支正好半个月的量。也可以只用一支——想抗初老修护就晚上用蓝色，想提亮就白天用橙色，但日夜配合着用是这套产品设计的完整逻辑。」 → 别把"可以单用"讲成"单用效果一样好"，配合使用才是产品设计的本意。
「敏感肌能用吗」「会不会过敏」
→
✗ 谨慎回答，讲清楚成分
「夜用含薰衣草油、白花春黄菊花油，日用含香精，敏感肌或者精油过敏体质建议先做局部测试，确认没问题再全脸用。这套的安全性和功效检测报告还在推进中，我不夸大讲，测试完成会第一时间更新。」 → 这款目前没有像面膜那样的第三方检测报告，别现编数据，坦诚讲"检测中"比瞎编更让成分党信任。
「多少钱」「什么时候上」
→
✗ 如实说未定

🌙
夜用十肽矩阵 · GHK-Cu + 经典胶原刺激肽
十种以上活性肽联用 · 成分党最爱开箱的一支
+
直播间钩子
「夜用这支成分表拉出来，十种以上活性肽——三肽-1铜也就是GHK-Cu，是护肤界研究了几十年的铜肽，促进胶原和弹性蛋白合成有大量文献；棕榈酰五肽-4是行业里最经典的胶原刺激肽之一，很多大牌抗老精华的核心就是它。」
粉丝追问：这么多肽会不会用不完/浪费？
「肽类是微量起效的成分，不是加得越多越浪费，配方里这么多种是覆盖不同修护方向——紧致、屏障、舒缓分工不同，晚上睡觉这段时间正好给皮肤做完整的修复工作。」
☀
日用三路抗氧化 · 凝血酸+四氢姜黄素+山竹果皮
白天焕活提亮 · 和LYMO系列成分同源
+
直播间钩子
「白天这支主打抗氧化提亮——凝血酸是成分党很熟悉的经典提亮成分，四氢姜黄素是姜黄素家族里更稳定的衍生物，山竹果皮提取物和黎沫系列松茸悬油精华里的倒捻子素是同一个抗氧化家族，自然源溯和LYMO虽然是两个牌子，但成分逻辑是相通的。」
粉丝追问：凝血酸不是止血的吗，护肤品里加安全吗？
「凝血酸（传明酸）外用在护肤品里是常见的提亮成分，和医用止血是两个使用场景和浓度，护肤品里的添加量是按化妆品法规来的，这个成分在成分党圈子里认知度很高。」
🧪
甘草酸二钾 · 贯穿自然源溯全系列
品牌一致性话题 · 和光甘草定面膜串着讲
+
直播间钩子
「甘草酸二钾这个成分，日用夜用两支都有，自然源溯的光甘草定面膜里也有甘草相关成分，甘草这条抗炎舒缓的线是贯穿整个品牌产品线的，不是单一产品的偶然添加。」
粉丝追问：甘草酸二钾和光甘草定是一个东西吗？
「不是同一个成分，但都来自甘草家族——甘草酸二钾偏抗炎舒缓，光甘草定偏提亮。自然源溯在好几个产品里都用了甘草相关成分，这是品牌一贯的配方思路。」
Part 04 · 退货风险

这几点
提前说清楚

这款还在筹备期，风险点更要提前说清楚

高风险
编造价格/检测数据
价格和第三方检测报告目前都没有确定，现场编数字或编报告，一旦对不上会砸口碑。
「价格和检测报告都在推进中，我现在讲的是成分逻辑，不是检测结果」——坦诚比编造更安全。
中风险
精油/香精过敏体质直接使用
夜用含薰衣草油、白花春黄菊花油，日用含香精，过敏体质直接全脸用可能刺激。
「首次用建议先局部测试」——提前说，减少过敏退货。
中风险
期待"一支顶两支"的效果
日夜精华是配合使用设计的，只用一支体验会打折，粉丝可能觉得"效果不够"。
「配合使用是这套产品设计的本意，只用一支也可以，但完整体验建议日夜都用」——预期讲清楚。
低风险
被问具体肽的功效数据
成分党可能追问某个肽的具体临床数据，这款目前没有产品级的检测报告。
「这几个肽在行业里研究比较久，文献可以自己查，我们产品本身的检测报告还在推进」——不要把行业公开文献讲成"我们自己测的"。
